# Professional Practice Report – Streamlit App

## Overview
This Streamlit application supports the analysis and visualisation of data for my Professional Practice report.  
It provides interactive dashboards and breakdowns to explore performance metrics and trends.

The app is deployed via Streamlit Cloud and hosted on GitHub.

---

## Features
- Interactive data visualisations
- Multi-page Streamlit layout
- Plotly charts for detailed breakdowns
- Clean, user-friendly interface
